# version-manager

With this library you can control your git versioning.

## How to install:
Global
```
npm install -g @be-tech/version-manager
```
Project as dev dependency
```
npm install @be-tech/version-manager --save-dev 
```

## How to use:
Just type 
Global install
```
v-manager 
```
Project as dev dependency
```
npm run v-manager
```
